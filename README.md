# Unreal AI NPC Dialogue

This project explores real-time AI driven NPC dialogue in Unreal Engine 5.

Current progress:

- Player can interact with NPC using Enhanced Input
- Dialogue UI system implemented with UMG
- Blueprint ↔ C++ event communication working
- AIChatComponent sends HTTP requests from Unreal
- Python FastAPI server connects to GLM-4-Flash
- NPC responses are streamed back into the game

Goal:
Create immersive AI-driven narrative NPCs with memory and personality.
